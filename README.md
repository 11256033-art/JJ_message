# JJ_messenger 期中分組作業

這是一個類似 Facebook Messenger 的 React Native 應用程式。
包含底部導覽 (Tab Layout)、聊天列表頁面、聊天室詳細內容，以及可以從手機相簿選擇照片的個人頭像設定 (Image Picker)。

## 功能說明
1. **Chat (聊天列表)**：顯示聯絡人名單，點擊聯絡人可進入聊天室查看訊息。
2. **Profile (個人設定)**：使用 `expo-image-picker` 從手機相簿選擇圖片並設定為個人頭像。

## 執行方式
```bash
npm install
npx expo start
